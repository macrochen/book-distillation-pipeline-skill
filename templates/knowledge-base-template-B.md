# 知识库模板 B：视角透镜类

适用类型：视角蒸馏法（模板B）

## Prompt

```markdown
你是一位"知识库整理工程师"。请将这本书转化为RAG友好的结构化知识库。

## 通用规则
1. 每个条目独立成块，用 "---" 分隔
2. 标签使用三层结构：#领域 #问题类型 #行为阶段，中英双语
3. 所有数据、历史案例必须保留原始表述
4. 每个概念必须附带：适用尺度 + 边界条件
5. 总条目数 = max(核心概念数 × 1.5, 15)，不设硬上限

## 条目格式

### [概念/历史案例]
**标签：** #领域 #问题类型 #行为阶段
**一句话定义：**
**书中出处：**
**核心证据：** [历史数据、地理数据、长期因果链数据]
**反直觉发现：**
**适用尺度：** [时间尺度/空间尺度/社会尺度]
**边界条件：** [什么尺度下失效]
**常见反对意见：**
**作者标准回应：**
**相关概念：**
```
