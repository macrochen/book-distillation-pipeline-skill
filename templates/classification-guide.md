# 书籍分类指南

## 决策树

使用以下决策树确定书籍的**主类型**：

```
Q1: 教你"遇到情况如何思考/决策"？
    ├─ 是 → 【三层蒸馏法】（模板A）
    │         例：《思考快与慢》《噪声》《助推》《反脆弱》
    │
    └─ 否 → Q2: 提供"看世界的独特透镜"？
              ├─ 是 → 【视角蒸馏法】（模板B）
              │         例：《枪炮、病菌与钢铁》《人类简史》
              │
              └─ 否 → Q3: 记录"高成就者的思维方式"？
                    ├─ 是 → 【人格模拟法】（模板C）
                    │         例：《富兰克林自传》《穷查理宝典》
                    │
                    └─ 否 → Q4: 训练"伦理/逻辑思辨"？
                          ├─ 是 → 【论证映射法】（模板D）
                          │         例：《公正》《道德情操论》
                          │
                          └─ 否 → Q5: 提供"可直接执行的技能流程"？
                                ├─ 是 → 【技能拆解法】（模板E）
                                │         例：《深度工作》《如何阅读一本书》
                                │
                                └─ 否 → 【案例-原则-行动法】（模板F）
                                          例：《坚毅》《刻意练习》
```

## 六种主类型说明

| 模板 | 类型 | 核心特征 | 典型书籍 |
|------|------|---------|---------|
| A | 三层蒸馏法 | 决策纠偏、认知心理学 | 《思考快与慢》《噪声》 |
| B | 视角蒸馏法 | 宏观透镜、结构性分析 | 《枪炮、病菌与钢铁》《人类简史》 |
| C | 人格模拟法 | 传记、人物思维方式 | 《富兰克林自传》《穷查理宝典》 |
| D | 论证映射法 | 哲学思辨、伦理张力 | 《公正》《道德情操论》 |
| E | 技能拆解法 | 实用方法论、可执行流程 | 《深度工作》《原子习惯》 |
| F | 案例-原则-行动法 | 故事驱动、原则提炼 | 《坚毅》《刻意练习》 |

## 辅模块补充

若书籍横跨多个类型，采用**「主模板 + 1-2个辅模块」**模式：

| 书籍 | 主类型 | 辅模块 |
|------|--------|--------|
| 《反脆弱》 | 三层蒸馏法 | 视角蒸馏法的"核心因果链" + "边界警告" |
| 《穷查理宝典》 | 三层蒸馏法 | 人格模拟法的"语音特征" + "禁忌清单" |
| 《原则》 | 技能拆解法 | 人格模拟法的"决策风格" + "场景回应库" |
| 《人类简史》 | 视角蒸馏法 | 论证映射法的"中立规则" |

**规则**：辅模块≤2个，禁止平铺所有类型。

```markdown

- **模板**: [模板X]

## 辅模块（如有）
- **模块1**: [从Y类型抽取的模块名称]
  - 用途：[为什么需要这个辅模块]
- **模块2**: [从Z类型抽取的模块名称]（可选）
  - 用途：[为什么需要这个辅模块]

## 提取规则
- [列出本书需要特别关注的提取规则]
```
