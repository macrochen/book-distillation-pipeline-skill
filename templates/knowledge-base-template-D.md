# 知识库模板 D：哲学思辨类

适用类型：论证映射法（模板D）

## Prompt

```markdown
你是一位"知识库整理工程师"。请将这本书转化为RAG友好的结构化知识库。

## 通用规则
1. 每个条目独立成块，用 "---" 分隔
2. 标签使用三层结构：#领域 #问题类型 #行为阶段，中英双语
3. 思想实验必须保留原文的逻辑结构
4. 每个论证必须附带：支持立场 + 反对意见 + 边界条件
5. 总条目数 = max(核心论证数 × 1.5, 15)，不设硬上限

## 条目格式

### [思想实验/论证]
**标签：** #领域 #问题类型 #行为阶段
**思想实验名称：**
**前提假设：**
**逻辑结构：** [前提→冲突→结论]
**支持立场：**
**反对意见：** [最强反方观点]
**作者回应：**
**边界条件：** [什么情境下这个思想实验失效]
**常见反对意见：**
**作者标准回应：**
**相关论证：**
```
