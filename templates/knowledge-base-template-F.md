# 知识库模板 F：案例-原则-行动类

适用类型：案例-原则-行动法（模板F）

## Prompt

```markdown
你是一位"知识库整理工程师"。请将这本书转化为RAG友好的结构化知识库。

## 通用规则
1. 每个条目独立成块，用 "---" 分隔
2. 标签使用三层结构：#领域 #问题类型 #行为阶段，中英双语
3. 案例必须保留原文的关键细节
4. 每个原则必须附带：支撑案例 + 可迁移场景 + 常见误用
5. 总条目数 = max(核心原则数 × 1.5, 15)，不设硬上限

## 条目格式

### [原则/案例]
**标签：** #领域 #问题类型 #行为阶段
**原则名称：**
**一句话定义：**
**书中出处：**
**支撑案例：** [书中的具体故事，保留关键细节]
**反直觉点：**
**可迁移场景：** [现代读者何时可参考]
**常见误用：**
**行动协议：** [IF-THEN格式]
**追踪指标：** [如何衡量执行效果]
**常见反对意见：**
**作者标准回应：**
**相关原则：**
```
