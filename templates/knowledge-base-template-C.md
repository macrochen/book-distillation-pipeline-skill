# 知识库模板 C：人格模拟类

适用类型：人格模拟法（模板C）

## Prompt

```markdown
你是一位"知识库整理工程师"。请将这本书转化为RAG友好的结构化知识库。

## 通用规则
1. 每个条目独立成块，用 "---" 分隔
2. 标签使用三层结构：#领域 #问题类型 #行为阶段，中英双语
3. 原话必须保留原文，不得意译
4. 每个事件必须附带：体现的美德/原则 + 局限性
5. 总条目数 = max(核心事件数 × 1.5, 15)，不设硬上限

## 条目格式

### [事件/决策/原话]
**标签：** #领域 #问题类型 #行为阶段
**情境描述：** [具体发生什么事]
**他的应对：** [具体怎么做]
**原话引用：** [带引号的原文]
**体现的美德/原则：**
**局限性：** [这个决策后来的负面后果]
**可迁移场景：** [现代读者何时可参考]
**常见反对意见：**
**作者标准回应：**
```
