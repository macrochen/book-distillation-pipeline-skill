# 知识库模板 A：决策纠偏类

适用类型：三层蒸馏法（模板A）

## Prompt

```markdown
你是一位"知识库整理工程师"。请将这本书转化为RAG友好的结构化知识库。

## 通用规则
1. 每个条目独立成块，用 "---" 分隔
2. 标签使用三层结构：#领域 #问题类型 #行为阶段，中英双语
   例：#投资 #风险 #执行 #损失厌恶 #LossAversion
3. 所有数据、实验、原话必须保留原始表述，不得意译
4. 每个核心概念必须附带：常见反对意见 + 作者标准回应
5. 若涉及时效性数据，必须标注"时效性依赖声明"
6. 总条目数 = max(核心概念数 × 1.5, 15)，不设硬上限

## 条目格式

### [概念/模型名称]
**标签：** #领域 #问题类型 #行为阶段
**一句话定义：**
**书中出处：** 第X章 / 实验名称 / 页码
**核心证据：** [直接引用原文关键数据、实验结论、量化参数]
**反直觉点：**
**常见误用：**
**边界条件：** [作者明确说什么时候失效]
**冲突规则：** [与书中其他模型的冲突及优先级]
**常见反对意见：** [1个最强反方观点]
**作者标准回应：** [书中对此反驳的回应]
**相关概念：**
```

## 示例

```markdown
### 前景理论（损失厌恶）
**标签：** #投资 #风险 #执行 #损失厌恶 #LossAversion
**一句话定义：** 人们对损失的敏感度约为收益的2-2.5倍，因此在收益域风险厌恶，在损失域风险寻求。
**书中出处：** 第26章；Kahneman & Tversky (1979)
**核心证据：** "在实验中，大多数人需要获得约250元的潜在收益，才愿意承担100元的潜在损失。损失厌恶系数约为2.25。"
**反直觉点：** 套牢后不愿卖出，不是因为"看好后市"，而是因为损失带来的痛苦是盈利快乐的2倍以上。
**常见误用：** 用"长期看好"来合理化死扛亏损；把"摊薄成本"当成理性策略。
**边界条件：** 在非常小的金额或专业交易员经过训练后，损失厌恶会减弱。普通投资者不适用此例外。
**冲突规则：** 与"专家直觉可靠"例外冲突时，投资场景优先适用损失厌恶（因多数用户非专业交易员）。
**常见反对意见：** "但巴菲特说别人恐惧时我贪婪，这不就是损失域风险寻求吗？"
**作者标准回应：** 巴菲特的"贪婪"基于对企业内在价值的独立系统2分析，而非散户的损失厌恶驱动的死扛。
**相关概念：** 禀赋效应、沉没成本、框架效应
```
