# 作者原声提取 Prompt

## 使用方法

将此Prompt与书籍PDF一起发送给AI，提取作者的语音特征。

## Prompt

```markdown
请从书中专门摘录以下内容，用于后续人格/语气校准：

1. **口头禅**：作者反复使用的短语
2. **反问句式**：作者挑战读者时常用的问句结构
3. **比喻习惯**：作者解释抽象概念时偏好的类比
4. **否定句式**：作者反对某种观点时的典型表达
5. **数据引用习惯**：精确到小数 vs 约数

## 输出格式

| 类型 | 原文摘录 | 出现位置（章节/页码） | 建议用途 |
|------|---------|---------------------|---------|
| 口头禅 | "..." | 第X章 | 身份定义/开场白 |
| 反问句式 | "..." | 第X章 | 标志性提问 |
| 比喻习惯 | "..." | 第X章 | 概念解释 |
| 否定句式 | "..." | 第X章 | 反模式禁令 |
| 数据引用 | "..." | 第X章 | 证据支撑 |

## 质量门

抽查至少3处，用PDF搜索（Ctrl+F）核实原文存在。

## 提取数量要求

每种类型至少提取3-5个样本，确保覆盖：
- 高频出现的（反复使用3次以上）
- 有代表性的（体现作者独特风格的）
- 可迁移的（能在AI回复中自然使用的）
```
